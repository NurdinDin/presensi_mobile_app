# presensi_mobile_app

Proyek ini merupakan proyek aplikasi presensi yang dibangun untuk kebutuhan sekolah dalam mencatat data presensi siswa otmatis. aplikasi ini dilengkapi dengan fitur geolokasi dan monitoring orang tua yang terhubung dengan bot telegram sebagai media penyampaian informasinya. Proyek dibangun menggunakan framework flutter dengan getx sebagai state management.

Lakukan perintah "flutter pub get" sebelum menjalankan program.
